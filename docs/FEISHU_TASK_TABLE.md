## 飞书导入步骤

1. 打开飞书 → 多维表格 → 新建
2. 点右上角 `···` → 导入 → CSV
3. 选择上面保存的 `tasks.csv` 文件
4. 导入后设置视图：按状态分组、按负责人筛选
5. 分享表格链接到团队群

---

## 优先级说明

| 等级 | 含义 |
|---|---|
| P0 | 必须完成，阻塞后续任务 |
| P1 | 本周内完成 |
| P2 | 可延后 |
